## About the Game ℹ️

The 2048 game is a single-player sliding tile puzzle game where the objective is to combine numbered tiles to reach the ultimate goal of a tile with the number 2048.

## Features 🌟

- **Core Game Logic**: Implements the fundamental mechanics of the 2048 game, including tile sliding, merging, and new tile generation.
- **Graphical User Interface**: A simple and intuitive GUI built with `tkinter` for visualizing the game board and interacting with it.
- **Keyboard Controls**: Play the game using the 'W', 'A', 'S', 'D' keys for up, left, down, and right movements, respectively.
- **Dynamic Tile Colors**: Tiles have different background and text colors based on their numerical value, enhancing visual feedback.
- **Win/Loss Detection**: The game detects when the player has won (by reaching the 2048 tile) or lost (when no more moves are possible).

## Tech Stack 💻

- **Language**: Python 3
- **GUI Library**: `tkinter`
- **Key Libraries**: `random`

## How to Play 🕹️

1. **Launch the Game**: Run the `start_game.py` script as described in the Installation section.
2. **Game Board**: A 4x4 grid will appear. Initially, two tiles with the number 2 will be placed randomly.
3. **Controls**: Use the following keys to slide the tiles:
   - **Up**: `W` key
   - **Down**: `S` key
   - **Left**: `A` key
   - **Right**: `D` key
4. **Gameplay**: When you slide tiles, adjacent identical tiles will merge into one tile with double the value. A new tile with the value 2 will appear in a random empty spot after each valid move.
5. **Objective**: Combine tiles to reach the 2048 tile.
6. **Winning**: You win when you successfully create a 2048 tile.
7. **Losing**: You lose when the board is full and no more valid moves can be made.
